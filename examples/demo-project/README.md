# Demo project

This directory intentionally keeps only lightweight, sanitized demo information.

The original local prototype validated:
- package parsing
- evidence-page construction
- 2+1 sub-agent orchestration
- review-loop concept
- formal-delivery conversion concerns

The public repo does not include:
- raw bid documents
- exported deliverables
- certificate images
- full evidence attachments

## Demo summary

A private-cloud tender prototype was used locally to validate:
- business/evidence sub-agent output quality
- technical solution sub-agent output quality
- manager-layer gate control
- independent review findings before final delivery

## What the prototype proved

1. single-entry manager experience is workable
2. 2+1 sub-agent orchestration is more realistic than one agent doing everything
3. evidence pages are necessary for controlled formal output
4. final delivery must be separated from internal drafts
